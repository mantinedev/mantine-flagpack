Installation

Install package:

yarn add mantine-flagpack

Then import styles at the root of your application:

import 'mantine-flagpack/styles.css';

Usage

All flags images are based on flagpack. Unlike react-flagpack package mantine-flagpack provides tree shakable cjs and ems bundles (unused flags will not be included in the bundle). Add components exported from the package also support Mantine style props. Some flag components have huge size (up to 20kb) and it is not recommended to import all flags in your application – import only flags that you need:

// Bad – you will have huge bundle size
import * as Flags from 'mantine-flagpack';

// Good – unused flags are removed from the production bundle
import { ESFlag, RUFlag } from 'mantine-flagpack';
import { ESFlag } from 'mantine-flagpack';

function Demo() {
  return <ESFlag w={100} radius="md" />;
}
